Task 2- Stock Portfolio Tracker 

📖 Description

The Stock Portfolio Tracker is a simple Python application that calculates the total investment value of a user's stock portfolio. The user enters a stock name and quantity, and the program uses a predefined dictionary of stock prices to calculate the total investment. The result is displayed on the screen and saved to a text file.

🚀Features

- User inputs stock name and quantity
- Uses a predefined dictionary of stock prices
- Calculates total investment value
- Displays the investment amount
- Saves the result in a text file
- Console-based application

🛠️Technologies Used

- Coding Python

📂 Project Structure
```
Stock_Portfolio_Tracker/
│── stock_portfolio.py
│── investment.txt
│── README.md
```

▶️How to Run

1. Open Pydroid 3 or any Python IDE.
2. Create a file named `stock_portfolio.py`.
3. Copy and paste the code.
4. Save the file.
5. Run the program.
6. Enter the stock name and quantity.
